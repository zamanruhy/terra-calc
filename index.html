Для подготовки проекта пришлите:

  • Габаритные размеры или план помещения
  • Назначение помещения
    (какая деятельность будет вестись внутри)
  • Планировку внутри
    (стеллажи, рабочие места, зоны погрузки и хранения)

Если хотите, прикрепите пожелания по требуемой освещенности, расположению и количеству светильников, способу установки, необходимому сроку эксплуатации.

при расчете освещения учтем нормы санпин по освещенности помещений

  • Для складов, цехов,
    кладовых, производственных помещений
  • Для офисов, школ, больниц, жилых помещений
  • Для парковок и прилегающих территорий

Стоимость проекта освещения – бесплатно.

  • Коммерческое предложение с 2-3 вариантами оборудования на выбор
  • Документы и техническую информацию по светильникам
